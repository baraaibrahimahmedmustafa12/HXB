Simple Decoder

Paste encoded text below and press Decode. Choose type or leave on Auto for the tool to detect (Base64 / Hex / ROT13 / Caesar / URL-safe).

Tip: you can also choose a file containing the encoded text using Choose file.

If the decoded data is binary (images, zip...), the tool will show a download link. Auto-detection prefers Hex when the input contains only hex chars and even length.
